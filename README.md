 About the Project

This is the official website for **Al-Reem Mandhi**, one of Kerala's most beloved Arabian Mandhi restaurant chains with multiple branches across the state. The site is built to reflect the brand's heritage — slow fire, deep flavor, and authentic Arabian tradition — through a rich, scroll-driven web experience.




## ✨ Features

- 🔥 **Immersive scroll animations** — storytelling-style sections that guide the user through the brand journey
- 🍽️ **Signature Menu Showcase** — highlights all Mandhi variants: Chicken, Mutton, Beef, Alfaham, Peri Peri, Honey Chilly
- ⭐ **Testimonials Carousel** — real Google reviews from verified customers across Kerala
- 📍 **Branch Directory** — dedicated `/branches` page listing all Al-Reem locations
- 📱 **Fully Responsive** — optimized for mobile, tablet, and desktop
- ⚡ **Edge-optimized** — deployed on Vercel's global CDN for fast load times

## 🚀 Getting Started

### Prerequisites

- Node.js `v18+`
- npm / yarn / pnpm / bun

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/alreem-mandhi.git
cd alreem-mandhi

# Install dependencies
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

The app supports hot-reload — edit `app/page.tsx` and changes reflect instantly.

### Production Build

```bash
npm run build
npm start
```
